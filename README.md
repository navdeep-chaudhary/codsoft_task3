# Rock-Paper-Scissors Game

## Description

A simple Command-Line Rock-Paper-Scissors game developed using Python. It allows users to play against the computer by choosing Rock, Paper, or Scissors. The game determines the winner based on standard rules, keeps track of scores, and allows users to play multiple rounds.

## Features

* User Chooses Rock, Paper, or Scissors
* Random Computer Selection
* Winner Determination
* Score Tracking
* Multiple Rounds
* User-Friendly Command-Line Interface

## Technologies Used

* Python 3
* Random Module

## How to Run

```bash
python rock_paper_scissors.py
```
